# author-service

Simple CRUD service for managing authors.

## Features

- Create author
- Get all authors
- Get author by ID
- Update author
- Delete author

## Example Author

```json
{
  "id": 1,
  "name": "Georgi Gospodinov",
  "bio": "Bulgarian writer, poet, and playwright.",
  "dateOfBirth": "1968-01-07"
}
```

## Tech Stack

- Java
- Spring Boot
- PostgreSQL
- Maven
